# 참빛교회 웹사이트 프로젝트 인수인계 문서

## 📌 프로젝트 개요

참빛교회를 위해 제작된 현대적이고 반응형 웹사이트입니다. 교인들과 방문자들이 교회 정보를 쉽게 접하고, 온라인으로 소통할 수 있는 플랫폼입니다.

**제작 기간**: 2025년 1월  
**제작자**: 자원봉사 개발자  
**라이선스**: MIT (무료 사용 및 수정 가능)

---

## 🎯 주요 기능

### 1. 홈페이지 (메인)
- **히어로 섹션**: 교회 대표 이미지와 환영 메시지
- **예배 시간 안내**: 주일예배, 새벽기도회 등 예배 일정
- **최신 설교**: 최근 설교 영상 및 말씀
- **커뮤니티 피드**: 교인들의 간증, 기도제목, 소식 공유
- **참여 안내**: 교회 참여 방법 및 연락처

### 2. 교회 소개 (/about)
- 교회 비전과 사명
- 담임목사 소개
- 교회 연혁
- 오시는 길

### 3. 참빛TV (/tv)
- 설교 영상 아카이브
- 예배 실황 영상
- 특별 집회 영상

### 4. 교육과 훈련 (/education)
- 성경공부 프로그램
- 제자훈련 과정
- 새가족 교육

### 5. 선교와 전도 (/mission)
- 국내외 선교 활동
- 전도 프로그램
- 선교 후원 안내

### 6. 사역과 섬김 (/ministry)
- 각 부서별 사역 소개
- 봉사 참여 방법
- 사역팀 안내

### 7. 교회 소식 (/news)
- 주보 및 공지사항
- 행사 일정
- 교회 뉴스

### 8. 나눔과 참여 (/participate)
- 헌금 안내
- 기도제목 신청
- 상담 신청

### 9. 온라인 콘텐츠 (/online)
- 온라인 예배
- 온라인 성경공부
- 디지털 자료실

### 10. 문화교실 (/culture)
- 문화 프로그램 안내
- 수강 신청
- 프로그램 일정

### 11. 예배 안내 (/worship)
- 예배 순서
- 찬양 가사
- 예배 참여 안내

### 12. 설교 말씀 (/sermon)
- 설교 텍스트
- 설교 노트
- 말씀 묵상

### 13. 회원 기능
- **로그인/회원가입** (/login, /register)
- 커뮤니티 게시물 작성
- 좋아요 및 댓글 기능

## 💻 기술 스택

### 프론트엔드 (사용자가 보는 화면)
- **React 18**: 최신 웹 프레임워크
- **TypeScript**: 안정적인 코드 작성
- **Vite**: 빠른 개발 환경
- **React Router**: 페이지 이동 관리
- **Axios**: 서버 통신

### 백엔드 (서버, 아직 미구현)
- **FastAPI**: Python 기반 고성능 서버
- **MariaDB**: 데이터베이스
- **JWT 인증**: 보안 로그인 시스템

### 배포
- **프론트엔드**: GitHub Pages (무료 호스팅)
- **백엔드**: 클라우드타입 또는 AWS (추후 구축 필요)

## 📂 프로젝트 구조

```
chambit/
├── frontend/                    # 웹사이트 화면
│   ├── src/
│   │   ├── components/         # 재사용 가능한 UI 조각들

│   │   │   └── community/     # 커뮤니티 게시물
│   │   ├── pages/             # 각 페이지 화면
│   │   │   ├── Home/          # 홈페이지
│   │   │   ├── About/         # 교회 소개
│   │   │   ├── TV/            # 참빛TV
│   │   │   └── ...            # 기타 페이지들
│   │   ├── hooks/             # 재사용 로직
│   │   ├── contexts/          # 전역 상태 (테마 등)
│   │   ├── api/               # 서버 통신 코드
│   │   ├── utils/             # 유틸리티 함수
│   │   └── styles/            # 공통 스타일
│   ├── public/                # 이미지, 아이콘 등
│   └── package.json           # 프로젝트 설정
│
└── backend/                    # 서버 (추후 구축)
    └── (아직 미구현)
```

## 🚀 설치 및 실행 방법

### 필요한 프로그램
1. **Node.js** (v18 이상) - [다운로드](https://nodejs.org/)
2. **Git** - [다운로드](https://git-scm.com/)
3. **코드 에디터** (VS Code 권장) - [다운로드](https://code.visualstudio.com/)

### 설치 단계

#### 1. 프로젝트 다운로드
```bash
# 터미널에서 실행
git clone [프로젝트 주소]
cd chambit
```

#### 2. 프론트엔드 설치 및 실행
```bash
cd frontend
npm install          # 필요한 패키지 설치 (최초 1회)
npm run dev          # 개발 서버 실행
```

#### 3. 브라우저에서 확인
- 주소: http://localhost:5173
- 자동으로 브라우저가 열립니다

### 배포 방법 (GitHub Pages)
```bash
cd frontend
npm run build        # 배포용 파일 생성
npm run deploy       # GitHub Pages에 배포
```

## 🐛 문제 해결 (트러블슈팅)

### 문제 1: npm install 실패
```bash
# 해결 방법
rm -rf node_modules package-lock.json
npm cache clean --force
npm install
```

### 문제 2: 포트 충돌 (5173 이미 사용 중)
```bash
# 다른 포트로 실행
npm run dev -- --port 3000
```

### 문제 3: 빌드 오류
```bash
# TypeScript 오류 무시하고 빌드
npm run build -- --mode production
```

### 문제 4: 모바일에서 레이아웃 깨짐
- 브라우저 캐시 삭제
- 하드 리프레시 (Ctrl + Shift + R)

```bash

npm run dev

npm run build

# 코드 검사

# 패키지 업데이트

npm run deploy
```

```
홈페이지: frontend/src/pages/Home/Home.tsx
```

