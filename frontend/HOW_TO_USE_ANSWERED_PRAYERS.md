# ✨ 응답의 전당 - 사용 방법

## 📍 응답 버튼 위치

### 홈 화면에서
1. 홈 화면 스크롤
2. 내가 작성한 기도 카드 찾기 (아직 응답 안된 기도)
3. 기도 카드 하단에 **"✨ 응답 등록"** 버튼이 표시됩니다
4. 위치: `기도 수 · 댓글 수 · ✨ 응답 등록`

### 기도 목록 페이지에서
1. 상단 메뉴에서 "기도 나눔" 클릭
2. 내가 작성한 기도 찾기
3. 기도 카드 하단에 **"✨ 응답"** 버튼이 표시됩니다

## 🎯 응답 등록 방법

### 1단계: 응답 버튼 클릭
```
┌─────────────────────────────┐
│ 은혜받은자  2주 전          │
│                             │
│ 취업을 위한 기도            │
│ 6개월간 구직 활동을...      │
│                             │
│ 🙏 기도하기  💬 댓글        │
│ 127명 · 23개 · ✨ 응답 등록 │ ← 여기 클릭!
└─────────────────────────────┘
```

### 2단계: 간증 작성
모달이 열리면:
- 기도 제목이 자동으로 표시됩니다
- 간증 내용을 작성합니다 (최대 500자)
- 예시 가이드를 참고하세요

```
간증 내용 작성 예시:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
기도 올린 지 2주 만에 원하던 회사에서 
연락이 왔습니다! 면접도 순조롭게 
진행되어 합격 통보를 받았습니다. 

하나님께서 정말 응답해주셨어요. 
함께 기도해주신 모든 분들께 
감사드립니다! 🙏
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3단계: 등록 완료
- "✨ 응답 등록" 버튼 클릭
- 성공 메시지 확인
- 기도 카드가 황금빛으로 변합니다!

## 🌟 응답된 기도 확인

### 홈 화면에서
응답된 기도는 다음과 같이 표시됩니다:

```
┌─────────────────────────────┐
│ ✨ 응답됨                   │ ← 배지 표시
│ 은혜받은자  2주 전          │
│                             │
│ 취업을 위한 기도            │ ← 황금빛 배경
│ 6개월간 구직 활동을...      │
│                             │
│ 🎉 간증                     │ ← 간증 섹션
│ 기도 올린 지 2주 만에...    │
│                             │
│ 🙏 127  💬 23              │
└─────────────────────────────┘
```

### 응답의 전당 페이지에서
1. 홈 화면 상단의 황금빛 배너 클릭
2. 또는 메뉴에서 "✨ 응답의 전당" 클릭
3. 모든 응답된 기도를 한눈에 확인

## 🎨 시각적 효과

### 황금빛 카드
- 그라데이션 배경 (황금색)
- 반짝이는 애니메이션
- "✨ 응답됨" 배지
- 간증 섹션 강조

### 응답의 전당 페이지
- 황금빛 배경
- 떠다니는 이모지 장식
- 특별한 헤더 디자인

## ❓ 자주 묻는 질문

### Q: 응답 버튼이 안 보여요
A: 다음을 확인하세요:
- 내가 작성한 기도인가요?
- 이미 응답 등록을 했나요?
- 로그인이 되어 있나요?

### Q: 응답을 수정하고 싶어요
A: 백엔드 연동 후 수정 기능이 추가될 예정입니다.

### Q: 다른 사람의 기도에도 응답을 등록할 수 있나요?
A: 아니요, 본인이 작성한 기도에만 응답을 등록할 수 있습니다.

### Q: 응답을 취소할 수 있나요?
A: 백엔드 연동 후 취소 기능이 추가될 예정입니다.
