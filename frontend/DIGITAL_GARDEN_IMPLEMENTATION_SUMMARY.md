## 💡 핵심 알고리즘

### 1. 나선형 꽃 배치
```typescript
const angle = index * (Math.PI * 2 * 3) / total // 3바퀴 나선
const radius = 30 + (index / total) * 60 // 중심에서 바깥으로
return {
  x: 50 + Math.cos(angle) * radius,
  y: 50 + Math.sin(angle) * radius,
}
```

### 2. 포인트 계산
```typescript
points = 
  (기도 횟수 × 2) +
  (구절 읽기 × 1) +
  (장 완독 × 3) +
  (책 완독 × 50) +
  (댓글 × 3) +
  (기도중 × 2)
```

### 3. 레벨 계산
```typescript
for (let i = LAMB_STAGES.length - 1; i >= 0; i--) {
  if (points >= LAMB_STAGES[i].minPoints) {
    return LAMB_STAGES[i]
  }
}
```
