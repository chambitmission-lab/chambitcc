# 설교 트랜스크립트 기능 빠른 시작 가이드

## 🚀 5분 안에 시작하기

### 관리자용: 트랜스크립트 업로드

1. **설교 등록**
   ```
   설교 목록 → "설교 등록" 버튼 → 정보 입력 → "등록하기"
   ```

2. **트랜스크립트 업로드**
   ```
   등록 완료 후 → "설교 내용 자동 생성" 체크 (선택) → "JSON 파일 선택" 버튼 → 파일 선택 → 자동 분석
   ```

3. **결과 확인**
   ```
   "X개의 성경 구절이 추출되었습니다" 메시지 확인
   "설교 내용이 자동으로 생성되었습니다" 메시지 확인 (옵션 선택 시)
   ```

### 사용자용: 성경 구절 탐색

1. **설교 보기**
   ```
   설교 목록 → 설교 카드 클릭 → 상세보기
   ```

2. **성경 구절 확인**
   ```
   아래로 스크롤 → "언급된 성경 구절" 섹션
   ```

3. **상호작용**
   - 🔊 보라색 타임스탬프 버튼 클릭 → 음성 플레이어 해당 시간으로 이동 및 자동 재생
   - ▶️ 빨간색 타임스탬프 버튼 클릭 → 비디오 해당 시간으로 이동
   - 📖 카드 클릭 → 성경 상세보기로 이동

## 📄 트랜스크립트 JSON 예시

```json
{
  "segments": [
    {
      "start": 168.5,
      "end": 172.3,
      "text": "오늘 말씀에 사무엘상 3장 3절 상반절에"
    }
  ]
}
```

## ✨ 주요 기능

- ✅ 자동 성경 구절 추출
- ✅ 설교 내용 자동 생성 (트랜스크립트 요약)
- ✅ 음성 플레이어 타임스탬프 연동 (우선)
- ✅ 비디오 타임스탬프 연동 (대체)
- ✅ 오디오 자동 재생 및 스크롤
- ✅ 성경 본문 미리보기
- ✅ 원클릭 성경 상세보기 이동

## 🎯 사용 팁

1. **트랜스크립트 품질**: 정확한 성경 구절 표현 사용 (예: "사무엘상 3장 3절")
2. **자동 요약**: 설교 내용이 비어있을 때 자동 생성 옵션 활용
3. **비디오 URL**: YouTube URL 형식 지원
4. **모바일**: 모든 기능이 모바일에서도 완벽하게 작동

## 📞 문제 해결

- **업로드 실패**: JSON 형식 확인, 관리자 권한 확인
- **구절 미추출**: 트랜스크립트 텍스트에 성경 구절 표현 포함 여부 확인
- **비디오 이동 안됨**: YouTube URL 형식 확인
