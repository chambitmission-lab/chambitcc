## 🎮 사용 시나리오

### 시나리오 1: 첫 정원 만들기
1. `/bible` 페이지 접속
2. 창세기 1장 선택
3. 음성 읽기 모드 활성화
4. 구절 읽기 (자동으로 꽃 폭죽 효과)
5. `/garden` 페이지에서 정원 확인
6. 🌹 장미가 피어있음!

### 시나리오 2: 어린 양 키우기
1. 기도하기 (2P)
2. 성경 읽기 (1P/절)
3. 댓글 작성 (3P)
4. 50P 달성 → 🐏 뛰어노는 양으로 레벨업!
5. 레벨업 축하 폭죽 효과 🎉

### 시나리오 3: 정원 가꾸기
1. 매일 성경 읽기
2. 다양한 책 읽기 (다양한 꽃 타입)
3. 월말에 정원 확인
4. "이번 달 정원은 이만큼 아름다워졌어요!" 🌸

## 📊 포인트 획득 방법

### 빠르게 포인트 모으기
```
1. 성경 1장 완독 (31절 기준)
   - 구절 읽기: 31P
   - 장 완독 보너스: 3P
   - 총: 34P

2. 기도 10회
   - 10회 × 2P = 20P

3. 댓글 5개 작성
   - 5개 × 3P = 15P

총 69P → 레벨 1 달성! 🎉
```

### 레벨별 목표
- **레벨 0 → 1**: 50P (성경 2장 정도)
- **레벨 1 → 2**: 100P (성경 3장 정도)
- **레벨 2 → 3**: 150P (성경 4-5장 정도)
- **레벨 3 → 4**: 200P (성경 6장 정도)

## 🌸 꽃 타입 가이드

### 어떤 꽃을 피울까?
```
🌹 장미를 원한다면: 창세기, 출애굽기 등 오경 읽기
🌸 백합을 원한다면: 시편, 잠언 등 시가서 읽기
🌷 튤립을 원한다면: 이사야, 예레미야 등 예언서 읽기
🌼 데이지를 원한다면: 마태복음, 요한복음 등 복음서 읽기
🌻 해바라기를 원한다면: 로마서, 고린도서 등 서신서 읽기
🌺 난초를 원한다면: 요한계시록 읽기
```

### 다채로운 정원 만들기
```
추천 읽기 순서:
1. 창세기 1장 (🌹 장미)
2. 시편 23편 (🌸 백합)
3. 요한복음 3장 (🌼 데이지)
4. 로마서 8장 (🌻 해바라기)

→ 4가지 색깔의 아름다운 정원!
```

## 🎉 보상 효과 체험

```

효과: 작은 꽃 폭죽
색상: 무지개 색 (6가지 꽃 색상)
```

### 2. 레벨업 폭죽
```
트리거: 어린 양 레벨업
효과: 3초간 양쪽에서 화려한 폭죽
```

### 3. 마일스톤 폭죽
```
트리거: 정원 10, 50, 100, 300, 500송이 달성
효과: 5초간 양쪽에서 지속적인 폭죽
```

## 🔍 문제 해결

### Q: 정원이 비어있어요
A: 성경 읽기 페이지에서 구절을 읽어보세요. 음성 읽기 모드를 활성화하고 구절을 읽으면 자동으로 추가됩니다.

### Q: 어린 양이 안 보여요
A: 프로필 페이지를 새로고침해보세요. 프로필 사진 아래에 표시됩니다.

### Q: 포인트가 안 올라가요
A: 활동 후 잠시 기다려주세요. React Query 캐시가 업데이트되는 데 몇 초 걸릴 수 있습니다.

### Q: 차트가 안 보여요
A: 최소 2일 이상의 데이터가 필요합니다. 성경을 더 읽어보세요!
