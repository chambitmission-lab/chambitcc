# ✨ 응답의 전당 - 빠른 시작 가이드

## 🚀 바로 사용하기

### 1. 응답의 전당 접근 방법

#### 방법 1: 홈 화면 배너
1. 앱 홈 화면으로 이동
2. '오늘의 말씀' 아래에 있는 황금빛 배너 클릭
3. "✨ 응답의 전당" 페이지로 이동

#### 방법 2: 상단 메뉴
1. 상단 햄버거 메뉴(☰) 클릭
2. 가로 스크롤 메뉴에서 "✨ 응답의 전당" 클릭

#### 방법 3: 직접 URL
```
/#/answered-prayers
```

### 2. 기도 응답 등록하기

#### 단계별 가이드
1. **내 기도 찾기**
   - 홈 화면 또는 기도 목록에서 내가 작성한 기도 확인
   - '내 기도' 배지가 표시됩니다

2. **응답 버튼 클릭**
   - 기도 카드 하단의 '✨ 응답' 버튼 클릭
   - (응답 버튼은 내가 작성한 기도에만 표시됩니다)

3. **간증 작성**
   - 모달이 열리면 간증 내용 작성
   - 최대 500자까지 입력 가능
   - 예시:
     ```
     - 구체적으로 어떤 일이 일어났나요?
     - 언제 응답을 받으셨나요?
     - 어떤 마음이 드셨나요?
     ```

4. **등록 완료**
   - '✨ 응답 등록' 버튼 클릭
   - 기도 카드가 황금빛으로 변합니다!

### 3. 황금빛 카드 효과 확인

응답된 기도는 다음과 같이 변합니다:

- 🌟 **황금빛 배경**: 그라데이션 효과
- ✨ **반짝이는 애니메이션**: 빛나는 효과
- 🎉 **간증 섹션**: 응답 내용 표시
- 🏆 **'응답됨' 배지**: 상단에 표시

## 🧪 테스트하기

### 실제 응답 등록 테스트
1. 로그인
2. 새 기도 작성
3. 작성한 기도에서 '✨ 응답' 버튼 클릭
4. 간증 작성 후 등록 → 백엔드 `POST /api/v1/prayers/{id}/answer` 호출
5. 황금빛 카드로 변환 확인 + 응답의 전당 페이지에 자동 노출

> 참고: `src/utils/mockAnsweredPrayers.ts`에 5개의 샘플 데이터가 남아 있지만,
> 현재는 어디서도 import되지 않으며 실제 백엔드 응답을 사용합니다.

## ⚠️ 현재 상태

### ✅ 완료된 기능
- 황금빛 카드 디자인 + 응답의 전당 페이지
- 응답 등록 / 수정 / 취소 모달
- **백엔드 응답 저장** — `POST /api/v1/prayers/{id}/answer`
- **응답 수정** — `PUT /api/v1/prayers/{id}/answer`
- **응답 취소** — `DELETE /api/v1/prayers/{id}/answer`
- **응답된 기도 필터링** — `GET /api/v1/prayers?is_answered=true`
- **푸시 알림** — 응답 등록 시 함께 기도한 사람들에게 자동 발송

## 💡 팁

1. **응답 버튼이 안 보여요**
   - 내가 작성한 기도인지 확인하세요
   - 이미 응답된 기도는 ✨ 응답 등록 대신 ✏️ 간증 수정 / ↩️ 응답 취소 버튼이 표시됩니다

2. **황금빛 효과가 멋져요!**
   - 응답의 전당 페이지에서 더 많은 응답 기도를 확인하세요
   - 반짝이는 애니메이션을 즐겨보세요

3. **간증을 수정하고 싶어요**
   - 응답된 본인 기도 카드의 ✏️ 간증 수정 버튼을 누르면 기존 내용이 채워진 모달이 열립니다
