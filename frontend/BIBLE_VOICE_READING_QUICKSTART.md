# 성경 음성 읽기 - 빠른 시작 가이드

## 🎯 기능 요약
사용자가 성경 구절을 직접 음성으로 읽어서 학습 완료를 체크하는 기능입니다.

## 🚀 사용 방법

### 1단계: 성경 읽기 페이지 접속
- 네비게이션에서 "성경 공부" 선택
- 원하는 책과 장 선택

### 2단계: 음성 읽기 모드 활성화
- 상단의 "음성 읽기 모드" 버튼 클릭
- 진행률 바가 표시됩니다

### 3단계: 구절 읽기
- 각 구절 옆의 마이크 버튼 클릭
- 구절을 소리내어 정확하게 읽기
- 자동으로 검증 및 읽음 처리

### 4단계: 진행 확인
- 읽은 구절은 체크 표시와 함께 흐리게 표시
- 상단 진행률 바에서 전체 진행 상황 확인

## ✨ 주요 특징

### 실시간 피드백
- 읽는 중: 음성 인식된 텍스트 실시간 표시
- 성공: "완벽합니다! 🎉" 등 긍정적 메시지
- 실패: "조금 더 정확하게 읽어주세요" 안내

### 유사도 검사
- 75% 이상 일치하면 통과
- 띄어쓰기, 구두점 무시
- 자연스러운 읽기 허용

### 진행률 추적
- 읽은 구절 / 전체 구절 표시
- 시각적 진행률 바
- 읽은 구절 하이라이트

## 💡 팁

### 효과적인 읽기
1. 조용한 환경에서 읽기
2. 명확하게 발음하기
3. 적당한 속도 유지
4. 원본 구절 그대로 읽기

### 문제 해결
- 인식 안 됨 → 마이크 권한 확인
- 계속 실패 → 더 정확하게 읽기
- 버튼 안 보임 → Chrome/Safari 사용

## 🔧 기술 스택
- Web Speech API (음성 인식)
- Levenshtein Distance (유사도 검사)
- React Hooks (상태 관리)
