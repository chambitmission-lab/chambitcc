# 🌸 디지털 가드닝: "기도로 키우는 어린 양" 가이드

## 📋 개요

사용자의 기도와 성경 읽기 활동을 시각적으로 보상하는 게임화 시스템입니다.
FaithTime, Forest 앱에서 영감을 받아 신앙 생활에 특화된 디지털 가드닝 기능을 구현했습니다.

## ✨ 주요 기능

### 1. 어린 양 캐릭터 시스템 🐑

사용자의 활동 포인트에 따라 성장하는 어린 양 캐릭터

#### 7단계 성장 시스템
| 레벨 | 이름 | 필요 포인트 | 이모지 | 특징 |
|------|------|-------------|--------|------|
| 0 | 갓 태어난 양 | 0 | 🐑 | 잠자는 애니메이션 |
| 1 | 뛰어노는 양 | 50 | 🐏 | 바운스 애니메이션 |
| 2 | 건강한 양 | 150 | 🐑✨ | 바운스 + 반짝임 |
| 3 | 튼튼한 양 | 300 | 🐏💪 | 점프 + 반짝임 |
| 4 | 지혜로운 양 | 500 | 🐑📖 | 점프 + 반짝임 |
| 5 | 빛나는 양 | 1000 | 🐏✨💫 | 떠다니기 + 글로우 |
| 6 | 천상의 양 | 2000 | 🐑👑✨ | 떠다니기 + 왕관 + 강한 글로우 |

#### 상태 메시지
- 0-50P: "기도가 필요해요 🙏"
- 50-150P: "말씀을 읽어주세요 📖"
- 150-500P: "건강하게 자라고 있어요! 💚"
- 500-1000P: "오늘도 함께해요! ✨"
- 1000P+: "당신의 신앙이 아름다워요 🌟"

### 2. 가상 정원 시스템 🌷

읽은 성경 구절이 꽃으로 피어나는 정원

#### 꽃 타입 (책별 매핑)
| 꽃 타입 | 이모지 | 색상 | 해당 책 |
|---------|--------|------|---------|
| 장미 | 🌹 | 빨강 | 오경 (창세기~신명기) |
| 백합 | 🌸 | 흰색 | 시가서 (욥기~아가) |
| 튤립 | 🌷 | 분홍 | 역사서, 예언서 |
| 데이지 | 🌼 | 노랑 | 복음서 (마태~요한) |
| 해바라기 | 🌻 | 주황 | 사도행전, 서신서 |
| 난초 | 🌺 | 보라 | 요한계시록 |

#### 정원 레벨
- 씨앗: 0송이
- 새싹 정원: 10송이
- 작은 화단: 50송이
- 아름다운 정원: 100송이
- 풍성한 정원: 300송이
- 천국의 정원: 500송이

#### 정원 통계
- 총 꽃 개수
- 정원 나이 (첫 꽃부터 경과 일수)
- 꽃 타입별 개수
- 가장 오래된/최신 꽃

### 3. 포인트 시스템 💎

활동별 포인트 획득

| 활동 | 포인트 |
|------|--------|
| 기도 1회 | 2P |
| 성경 구절 읽기 1절 | 1P |
| 성경 장 완독 1장 | 3P (보너스) |
| 성경 책 완독 1권 | 50P (보너스) |
| 댓글 작성 1개 | 3P |
| 기도중인 항목 1개 | 2P |

### 4. 보상 효과 🎉

#### Confetti 효과
- **레벨업**: 3초간 양쪽에서 폭죽
- **업적 달성**: 화려한 폭죽 쇼
- **꽃 피어남**: 작은 꽃 폭죽
- **정원 마일스톤**: 5초간 양쪽 폭죽
- **연속 기도**: 이모지 폭죽 (🔥⭐✨)

### 5. 성장 차트 📊

Recharts를 활용한 시각화

- **영역 차트**: 누적 구절 수 (그라데이션)
- **선 차트**: 누적 구절 수 (선)
- **막대 차트**: 일별 읽은 구절 수

최근 30일 데이터 표시

## 🚀 사용 방법

### 1. 정원 페이지 접속

네비게이션 메뉴에서 "🌸 나의 정원" 클릭 또는 `/garden` 경로로 이동

### 2. 어린 양 확인

- 현재 레벨과 상태 확인
- 다음 단계까지 필요한 포인트 확인
- 진행률 바로 시각적 확인

### 3. 정원 가꾸기

- 성경을 읽으면 자동으로 꽃이 피어남
- 꽃을 클릭하면 해당 구절 확인 가능
- 월별로 정원 히스토리 확인

### 4. 성장 기록 확인

- 차트 타입 선택 (영역/선/막대)
- 일별 읽은 구절 수 확인
- 누적 성장 추이 확인

export const calculateActivityPoints = (activity: UserActivityData): number => {
  let points = 0

// utils/gardenCalculator.ts
const generateFlowerPosition = (index: number, total: number) => {
  const angle = index * (Math.PI * 2 * 3) / total // 3바퀴 나선
  const radius = 30 + (index / total) * 60 // 중심에서 바깥으로
  return {
    x: 50 + Math.cos(angle) * radius,
    y: 50 + Math.sin(angle) * radius,
  }
}
