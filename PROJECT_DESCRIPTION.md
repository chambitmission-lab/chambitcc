# 참빛CC (Chambit Community Connect)

> 기도, 성경 읽기, 설교, 이벤트를 하나로 연결하는 교회 디지털 커뮤니티 플랫폼

## 📖 프로젝트 소개

참빛CC는 현대 교회 공동체를 위한 종합 디지털 플랫폼입니다. 단순한 교회 웹사이트를 넘어, 교인들이 일상에서 신앙 생활을 실천하고, 서로 기도로 연결되며, 영적으로 성장할 수 있도록 돕는 인터랙티브 커뮤니티 공간입니다.

### 🎯 핵심 가치

- **연결(Connect)**: 교인들을 기도와 신앙으로 연결
- **성장(Grow)**: 게이미피케이션을 통한 영적 성장 동기 부여
- **참여(Engage)**: 능동적인 신앙 생활 실천
- **공유(Share)**: 은혜와 간증을 나누는 커뮤니티

## ✨ 주요 기능

### 🙏 기도 커뮤니티
- **기도 제목 공유**: 개인의 기도 제목을 교회 공동체와 나눔
- **함께 기도하기**: 다른 교인의 기도 제목에 동참
- **기도 그룹**: 소그룹별 기도 제목 관리
- **응답된 기도**: 하나님의 응답을 간증으로 공유
- **기도 타이머**: 집중 기도 시간 관리 (Prayer Focus)
- **성경 구절 연결**: 기도 제목에 관련 성경 구절 첨부

### 📖 성경 공부
- **온라인 성경**: 한글/영문 성경 읽기
- **성경 검색**: 키워드로 구절 찾기
- **음성 읽기**: TTS 기능으로 성경 듣기
- **성경 읽기 계획**: 체계적인 통독 관리
- **구절 하이라이트**: 중요한 구절 표시 및 메모

### 🎤 설교 아카이브
- **설교 영상/음성**: 주일 설교 다시 보기/듣기
- **설교 원고**: 텍스트로 설교 내용 확인
- **성경 구절 참조**: 설교 중 인용된 성경 구절 바로 보기
- **설교 검색**: 주제별, 날짜별 설교 찾기
- **오디오 녹음**: 관리자용 설교 녹음 기능

### 🌱 디지털 정원 (Virtual Garden)
- **영적 성장 시각화**: 신앙 활동을 나무 성장으로 표현
- **나무 키우기**: 기도, 성경 읽기, 참여로 나무 성장
- **정원 꾸미기**: 개인화된 영적 정원 디자인
- **성장 차트**: 신앙 생활 통계 및 분석
- **어린양 캐릭터**: 귀여운 캐릭터와 함께하는 신앙 여정

### 🏆 업적 시스템
- **레벨 시스템**: 활동에 따른 레벨 업
- **배지 획득**: 다양한 신앙 활동 업적 달성
- **경험치(XP)**: 기도, 성경 읽기, 댓글 등으로 XP 획득
- **프로필 꾸미기**: 업적 배지로 프로필 개성화
- **동기 부여**: 게이미피케이션으로 지속적인 참여 유도

### 📅 교회 일정 관리
- **이벤트 캘린더**: 교회 행사 일정 한눈에 보기
- **참석 신청**: 온라인으로 행사 참석 등록
- **알림 기능**: 중요한 행사 리마인더
- **행사 댓글**: 행사에 대한 질문과 소통

### 👥 소그룹 관리
- **그룹 생성**: 구역, 셀, 팀별 그룹 구성
- **그룹 기도**: 그룹 전용 기도 제목 공유
- **그룹 활동**: 그룹별 성경 공부 및 모임 관리

### 📱 PWA (Progressive Web App)
- **앱처럼 사용**: 홈 화면에 추가하여 앱처럼 실행
- **오프라인 지원**: 인터넷 없이도 일부 기능 사용
- **푸시 알림**: 중요한 소식 실시간 알림
- **빠른 로딩**: 최적화된 성능

### 🔔 알림 시스템
- **실시간 알림**: 기도 응답, 댓글, 좋아요 알림
- **푸시 알림**: 모바일 푸시 알림 지원
- **알림 설정**: 개인화된 알림 관리

### 🎨 사용자 경험
- **다크 모드**: 눈의 피로를 줄이는 다크 테마
- **반응형 디자인**: 모바일, 태블릿, PC 모두 최적화
- **다국어 지원**: 한국어/영어 지원
- **접근성**: 모든 사용자를 위한 웹 접근성 고려

## 🎮 게이미피케이션 요소

### 경험치(XP) 획득
- 기도 제목 작성: 10 XP
- 다른 사람 기도 제목에 기도하기: 5 XP
- 성경 읽기: 15 XP
- 설교 듣기: 20 XP
- 댓글 작성: 3 XP
- 출석 체크: 5 XP

### 레벨 시스템
- 레벨 1: 새싹 (0-100 XP)
- 레벨 2: 묘목 (100-300 XP)
- 레벨 3: 어린 나무 (300-600 XP)
- 레벨 4: 성장한 나무 (600-1000 XP)
- 레벨 5: 큰 나무 (1000+ XP)

### 업적 배지
- 🙏 기도 전사: 100번 기도하기
- 📖 말씀 사랑: 성경 30일 연속 읽기
- 💬 격려자: 댓글 50개 작성
- 🌟 충성된 종: 출석 100일 달성
- 🎯 집중 기도: Prayer Focus 10회 완료

## 🚀 시작하기

### 설치

```bash
# 저장소 클론
git clone https://github.com/chambitmission-lab/chambitcc.git

# 프론트엔드 디렉토리로 이동
cd chambitcc/frontend

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 환경 변수 설정

```bash
# frontend/.env.development
VITE_API_BASE_URL=http://localhost:8000/api/v1
VITE_APP_NAME=참빛CC
```

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 📱 사용 시나리오

### 시나리오 1: 아침 묵상
1. 앱을 열고 오늘의 말씀 확인
2. 성경 읽기 계획에 따라 오늘의 본문 읽기
3. 은혜받은 구절에 하이라이트 표시
4. 묵상 내용을 기도 제목으로 작성
5. 디지털 정원의 나무가 성장 🌱

### 시나리오 2: 기도 제목 나누기
1. 기도 제목 작성 (개인적인 고민)
2. 관련 성경 구절 첨부
3. 소그룹 멤버들이 함께 기도
4. 응답받은 후 간증 업데이트
5. 커뮤니티에 은혜 나눔 ✨

### 시나리오 3: 주일 설교 복습
1. 주일 설교 영상 다시 보기
2. 설교 중 인용된 성경 구절 확인
3. 은혜받은 부분 메모
4. 설교 내용을 기도 제목으로 적용
5. 소그룹에서 나눔 주제로 활용 📖
